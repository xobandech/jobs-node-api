How to run project.<br />

In node-api-server .env set up MONGO_URI and JWT_SECRET variables <br />

in frontend .env set up DANGEROUSLY_DISABLE_HOST_CHECK=true and JWT_SECRET variables <br />

cd node-api-server <br />
npm i <br />
npm run start <br />
cd ../frontend <br />
npm i <br />
npm start <br />
